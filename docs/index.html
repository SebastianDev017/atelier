Theme documentation

Atelier

Editorial minimalist Shopify theme for considered objects, interior design, lifestyle, and premium goods.

Getting started

Installation

  • Purchase Atelier from the Shopify Theme Store
  • Go to Online Store → Themes → Add theme
  • The theme installs automatically with demo content
  • Click Customize to open the theme editor

Demo store setup

  • Add at least 6 products with professional images
  • Create collections and assign products to them
  • Configure your navigation menu under Online Store → Navigation
  • Create an About page using the "About page" template

Theme presets

Atelier includes three distinct presets accessible under Theme Settings → Theme style.

Atelier (default)

Warm cream palette (#FAF7F2) with espresso typography. EB Garamond headings, Lora body. No uppercase. Designed for interior design studios and home goods.

Studio

Near-white canvas (#F8F8F6) with high-contrast typography. Libre Baskerville headings, DM Sans body. Uppercase headings. For creative studios and photographers with a shop.

Maison

Warm linen palette (#F2EDE6) with gold accents. Playfair Display headings, Jost body. No uppercase. For premium fashion and lifestyle brands.

Theme settings

Typography

Change heading and body fonts using Shopify's font picker. Adjust heading scale (75–150%) and body scale (80–120%) to fine-tune type sizes across the theme.

Colors

Set background, text, accent, button, and border colors independently. Extended palette options for heading color, body color, muted/label color, and link hover color.

Images

Apply border-radius (0–24px) to all images globally. Enable an image border with configurable width and color.

Animations and experience

Toggle smooth scroll (Lenis), page transitions, custom editorial cursor, grain texture overlay, scroll reveal animations, and product image hover swap. Adjust animation speed (fast/normal/slow) and grain intensity.

Product page features

  • Notify me — When a product is sold out, a "Notify me" button replaces Buy Now. Opens an on-brand dialog with email subscription and consent checkbox.
  • Trust badges — Four configurable trust icons below the purchase buttons.
  • Collapsible tabs — Shipping, returns, and product details in accordion format.
  • Product recommendations — Shopify-powered complementary products.

Accessibility

Atelier is built to WCAG 2.2 AA standards. All interactive elements have aria-labels, focus indicators, and minimum 44×44px touch targets. Screen reader announcements are implemented for cart updates, price changes, and form feedback.
